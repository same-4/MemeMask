# MemeMask
**Mask the web with memes.**

## About
MemeMask is a Chrome extension that replaces all images on a webpage with a single meme image, creating a fun and chaotic browsing experience.

It uses JavaScript DOM manipulation through a Chrome content script to dynamically override images on supported websites.

## Features
- Replaces all images with a meme image
- Runs automatically on page load
- Lightweight and fast
- No data collection

## Tech Stack
- JavaScript
- Chrome Extension APIs (Manifest V3)
- DOM Manipulation

## Installation
1. Clone the repository
2. Go to `chrome://extensions`
3. Enable **Developer mode**
4. Click **Load unpacked**
5. Select the project folder
